Experimenting with React and ES6.

- React for views.
- Flux architecture inspired by Reflux.
- ES6 syntax with Babel.
- Building with Webpack.
- Tests with Mocha and Jsdom.
- Code style with JSCS.

## Getting started

- `npm install -g grunt-cli`
- `npm install`
- `grunt dev`
- Open `localhost:8080`
